- **Java Development Kit (JDK)**: 8 or higher
- **Build Environment**: Standard Java compiler (javac)
- **IDE**: VS Code, IntelliJ IDEA, Eclipse, or NetBeans (optional)

## 🚀 Quick Start

### Compile a single file:
```bash
javac src/codeArena/PrimeNo.java
```

### Run a specific program:
```bash
java -cp src codeArena.PrimeNo
```

### Compile all files in a package:
```bash
javac -d . src/codeArena/*.java
```

### Run from compiled classes:
```bash
java -cp . codeArena.CodeArena1
```
